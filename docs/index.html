Quick guides:

Installing plugins via seizuRepo
--------------------------------
1) Open Kodi and go to -> File Manager -> Add source -> https://seizu.github.io/seizuRepo
2) Go to AddOns -> Install from zip file -> choose seizuRepo -> Install "repository.seizu-x.x.zip"
3) Go to AddOns -> Install from repository -> choose seizuRepo -> Video add-ons -> Install your favorite plugin

Installing the FilmpalastEx plugin via Zip
------------------------------------------
1) Open Kodi and go to File Manager -> Add source -> https://seizu.github.io/seizuRepo
2) Go to AddOns -> Install from zip file -> choose seizuRepo -> Install "script.module.resolveurl-5.x.x.zip"
3) Go to AddOns -> Install from zip file -> choose seizuRepo -> Install "plugin.video.filmpalast.ex-0.x.x.zip"

Installing a Kodi custom splash screen
--------------------------------------
1) Go to "File manager" and add the following two sources:
    special://home
    https://seizu.github.io/seizuRepo   
2) Open 'home' you have just created in the left-hand window and go to the 'media' directory.   
3) Do the same with "seizuRepo" in the right-hand window and open the "splash_screen_collection.zip".      
4) Select your favorite splash screen image on the right side and copy it into the left media directory. 
   Use the right mouse button or the hamburger menu button to copy the image.  
5) Remove the digits in the file name. If the image is a jpg-file, rename it to "Splash.jpg", 
   if it is a png-file, rename it to "splash.png". !! Note that the name is case-sensitive !!
   e.g. "Splash21.jpg" to "Splash.jpg"
        "splash02.png" to "splash.png"
6) Close and reopen Kodi to test your splash screen.
